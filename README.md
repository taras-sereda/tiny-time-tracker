# tiny-time-tracker
minimalistic time tracker
1. Track: 

    ```
    # real-time mode
    python track_time.py research tts

    # off-line mode
    python track_time.py research tts --duration 8 --date 18-10-2023

    ```

2. Create report:

    ```
    python write_report.py --data-dir user-data/10-2023

    ```
